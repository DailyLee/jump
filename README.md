## 源码目录介绍
```
./js
├── base                                   // 定义游戏开发基础类
│   ├── god.js                             // three.js的简易分装
│   └── router.js                          // 游戏路线类
├── libs
│   ├── symbol.js                          // ES6 Symbol简易兼容
│   ├── three.js                           // 游戏引擎
│   ├── twweb.js                           // 动画插值
│   └── weapp-adapter.js                   // 小游戏适配器
│   └── controls/orbitcontrols.js          // 视图变化控制器
├── elements
│   └── chess.js                           // ♟棋子
│   └── desk.js                            // 桌子
│   └── floor.js                           // 地板
│   └── elementPool.js                     // 对象池简易实现
└── main.js                                // 游戏入口主函数

```
